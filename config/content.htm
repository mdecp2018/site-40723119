Github常用基本語法

git help 可以查詢所有git指令

git init

建立一個空的git倉儲

git add

為我當前的資料作暫存

git add . 對所有的資料作暫存

gti diff

檢視遠端與近端的差異

git commit -m"xxx"

在近端提交我的改版內容，並稱做xxx

git push

將我的改版內容送到遠端

git log

檢視提交過的資料

git checkout 位置資料前七碼

到我所指定的資料位置

git status

查看所有檔案的狀態

git branch

查詢我所在的分支

git branch xxx 創建一個新的xxx分支

git pull

將遠端拉到近端

git remote add xxx https:--------------

新建一個遠端位置

git remote

檢視遠端資料

git merge

合併分支

git clone

將遠端的資料下載到近端

git version

檢視當前版本
